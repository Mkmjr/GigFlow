## Section 1 – Product Vision
Gigflow is a product based web application which serves as a platform for providing and posting freelance projects.
Gigflow helps client in posting the desired projects and their reasonable amount for completion of the project, whereas freelancers can pick up the desired project as per their capability and skills, earn money and refer.
Gigflow has been created as a portfolio project that showcases all the modern prodcut based team principle and architecture been used to build a project.
